# <a name="send-feedback"></a>Enviar comentários

Seus comentários ajudam a melhorar a Pesquisa da Microsoft. Em um resultado de trabalho, use o link Comentários para fazer comentários sobre o que você gostou, não gostou ou para sugerir novos recursos ou resultados que você gostaria de adicionar. Você também pode usar o link para reportar qualquer problema que tiver.
  
## <a name="send-feedback-about-work-results"></a>Envie comentários sobre resultados de trabalho

Para enviar comentários, na parte inferior da qualquer resultado de trabalho, clique em **Comentários**. Dependendo do tipo de resultado, você poderá ver opções para enviar comentários à Microsoft ou ao departamento de TI.
  
Os comentários que você enviar para a Microsoft são postados no nosso widget UserVoice. Qualquer pessoa que enviar comentários poderá ler e se subscrever aos seus comentários.
  
Os comentários que você envia para o departamento de TI não são compartilhados diretamente com a Microsoft.
  
## <a name="send-feedback-about-web-results"></a>Envie comentários sobre resultados da web

Conforme você usa a Pesquisa da Microsoft, talvez precise fazer comentários sobre os resultados da web exibidos. Para enviar esses comentários, clique em **Comentários** na parte inferior de qualquer página do Bing. Esses comentários não são compartilhados com a sua organização. O Bing revisa todos os comentários, mas você não receberá uma resposta pessoal.
