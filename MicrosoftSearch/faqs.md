# <a name="faqs"></a>Perguntas frequentes

Estas são as perguntas frequentes que usuários e organizações têm sobre como administrar e usar o Microsoft Search.
  
## <a name="whats-the-value-proposition-for-microsoft-search"></a>Qual é a proposta de valor para o Microsoft Search?

Microsoft Search é uma maneira segura para pesquisar seu trabalho e o conteúdo da web. Esse tipo de integração entre web e enterprise só está disponível com a Microsoft. Para obter informações relacionadas à conformidade, consulte [Security for Microsoft Search](security.md).
  
## <a name="what-microsoft-search-features-are-available-now"></a>Quais recursos do Microsoft Search estão disponíveis agora?

Para obter uma lista completa, consulte [Recursos do Microsoft Search](features.md).
  
## <a name="does-microsoft-search-support-advanced-query-understanding"></a>Significa compreensão de consulta do Microsoft Search suporte Avançado?

Sim, o Microsoft Search analisa a intenção de consulta de frases maiores. Esse recurso usa AI para aprender a usuários de supérfluo frases comuns adicionarem para as consultas que não afetam seu intenção de pesquisa. Por exemplo, quando um usuário procura 'conte-me mais sobre como alterar minha senha por favor,' podemos extrair as palavras menos importantes da consulta e o gatilho com base nos relevantes like 'Alterar senha'.
  
Esse recurso não substituirá o conjunto de palavras-chave no portal de administração.
  
## <a name="does-microsoft-search-search-for-files-on-premises-as-well-as-the-cloud"></a>Microsoft Search procura por arquivos no local, bem como a nuvem?

Oferece suporte ao Microsoft Search procurando documentos live no SharePoint Online, OneDrive for Business, bem como sobre híbrido local SharePoint e o SharePoint Online para os tipos de arquivo mais comuns do Office. Se você pode pesquisar conteúdo local no SharePoint Online, você poderá encontrar conteúdo local usando o Microsoft Search. 
  
## <a name="does-microsoft-search-replace-other-enterprise-search-experiences"></a>Microsoft Search substituir a outras experiências de pesquisa da empresa?

Microsoft Search é uma oferta complementar que agrupa os resultados de várias fontes. Para obter mais informações sobre fontes e tipos de arquivos suportados, consulte o próximo FAQ.
  
## <a name="what-file-types-and-sources-does-microsoft-search-support"></a>Quais tipos de arquivo e fontes é compatível com o Microsoft Search?

Oferecemos suporte as fontes de conteúdo a seguintes:
  
- SharePoint Online
    
- Híbrido do SharePoint (local + SPO)
    
- OneDrive for Business
    
Os seguintes tipos de arquivo são exibidos na pesquisa de arquivo e apareçam na guia arquivos para pessoas e grupos:
  
- Word
    
- Excel
    
- PowerPoint
    
- OneNote
    
- PDF

## <a name="how-does-microsoft-search-use-azure-active-directory"></a>Como o Microsoft Search usar o Windows Azure Active Directory?

Microsoft Search está de acordo com a identidade manipulação standard práticas em todos os produtos da Microsoft. Isso significa que seus usuários podem ser automaticamente conectados por meio do serviço single sign-on.
